# student_performance_prediction_using_fastapi
## 📌 Overview
This project predicts student exam performance using Regrssion model.

## 🚀 Features
- Data preprocessing & EDA
- Regression model training
- FastAPI deployment
- Real-time prediction API

## 🛠️ Tech Stack
- Python
- Scikit-learn
- FastAPI

## 📊 Model
- Algorithm: Linear Regression
- Accuracy: 99%

## 📡 API Endpoint
POST /predict

Example Input:
[5, 7, 80]
